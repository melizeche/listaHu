Documentación (en proceso)

En este momento el API solo está habilitado para consulta, mas adelante se va a poder agregar nuevas denuncias entre otras cosas.

Endpoints

GET /denuncias/

http://listahu.org/api/v1/denuncias

Se pude filtrar por numero, tipo[spam,estafa,extorsion], id y fecha

Ejemplos

  • https://listahu.org/api/v1/denuncias/?numero=595975849399
  • https://listahu.org/api/v1/denuncias/?added_from=2015-03-01&added_to=2015-03-20
  • https://listahu.org/api/v1/denuncias/?id_from=100&id_to=200
  • https://listahu.org/api/v1/denuncias/?tipo=extorsion
