¿Qué es?

Lista Hū es un proyecto que nace con el fin de crear una base de datos de números telefónicos que realizan llamadas o envían mensajes no deseados. Estos mensajes y/o llamadas son usualmente de extorsión, estafa o publicidad invasiva. El objetivo es identificar colectivamente a estos números para su posterior bloqueo desde una lista negra o lista hū.

La idea

Todos los días vemos en las distintas plataformas de noticias una cantidad abrumadora de personas que reciben llamadas y/o mensajes de tinte extorsivo (secuestros virtuales) o no deseado. Muchas de estas personas son estafadas en su buena fe o son víctimas de violación de privacidad desde la venta de bases de datos con sus números telefónicos.

Lista Hū tiene por objetivo utilizar el recurso más importante de una comunidad, la colaboración entre sus miembros, para poder generar una base de datos que permita primero identificar y luego bloquear a los números que realizan este tipo de comunicaciones.

Actualmente se puede descargar la lista hū y consultar los números que están en ella, pero en un futuro cercano y con ayuda de la comunidad se pretende crear una app que se actualice automáticamente con los datos de la lista negra para bloquear así a estos números y reducir el riesgo potencial de recibir este tipo de comunicaciones.

Para colaborar con la lista hū basta con que llenes los campos del formulario y adjuntes una captura de pantalla del mensaje o, si se trata de una llamada, una captura del registro telefónico de tu celular en donde aparezca el número a ser denunciado.

Ejemplos

Descargá la Lista Hũ

Podés bajarla directamente a tu celular en un formato de archivo de contactos (vCard) para poder identificar si es que la llamada o mensaje que recibís pertenece a un remitente que ya fue denunciado por la comunidad. También está disponible en formato .csv (compatible con Excel) para ver los datos de la lista.

La documentación del API para desarrolladores está en proceso pero ya podés consultar la lista completa a través de un API RESTful que devuelve los datos en JSON.

A futuro

Además del desarrollo de una app para bloqueo automático de números denunciados por la comunidad se trabajará en un filtro de la lista por tipo de denuncia (números extorsivos, números de SPAM comercial, etc.), también un sistema de calificación de cada denuncia para validar los datos denunciados y la capacidad de descargar y bloquear solamente los números denunciados y verificados por la misma comunidad con sus votos.

Si tu número figura en la Lista Hū y crees que no debería estar allí por favor utiliza el formulario de contacto para solicitar su baja.
